Cara Menggunakan Aplikasi:

  1. Siapkan file cp-pai-sd.csv yang berisi Tujuan Pembelajaran (TP).
  2. Klik tombol "Upload File CSV" (warna oranye) di menu sebelah kiri.
  3. Pilih Kelas dan Semester. Data Bab/Elemen akan otomatis muncul.
  4. Pilih Bab untuk digenerate menjadi Modul Ajar dan LKPD.
  5. Lengkapi data sekolah, guru, dan rombel. Sistem akan menyusun Prota, Promes, Modul Ajar, KKTP, dan LKPD secara otomatis.
  6. Klik angka pada tabel KKTP untuk mengedit. Nilai rata-rata KKTP akan berubah otomatis!
  7. Klik tombol Cetak / Simpan PDF.

Aturan / Format Membuat File CSV:

  • Gunakan Microsoft Excel.
  • Buat 5 kolom dengan urutan persis seperti ini:
    Kolom A: Fase (contoh: Fase A)
    Kolom B: Kelas (contoh: 1)
    Kolom C: Semester (contoh: Ganjil)
    Kolom D: Elemen/Bab (contoh: Al-Qur'an Hadis)
    Kolom E: Tujuan Pembelajaran (contoh: Peserta didik mampu melafalkan...)
  • JANGAN gunakan koma (,) di dalam isi teks jika Anda menyimpannya dengan format standar.
  • Saat di-Save, pilih format CSV (Comma delimited) (*.csv) atau CSV (MS-DOS). Pemisah (delimiter) yang dibaca sistem ini adalah titik koma (;).
