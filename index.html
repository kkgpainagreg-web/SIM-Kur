1. Format CSV (8 Kolom)

Pastikan file Google Sheets/Excel Anda memuat 8 kolom berurutan: Mapel, Fase, Kelas, Semester, Bab, CP, TP, Profil Lulusan.

2. Cara Menambah Banyak Mapel Sekaligus

Anda TIDAK PERLU membuat link CSV berbeda untuk tiap Mapel. Cukup gunakan 1 file Excel, tambahkan baris ke bawah, dan ketikkan nama Mapel yang berbeda di Kolom A (Mata Pelajaran). Aplikasi akan otomatis membaca dan membuat Dropdown Pilihan Mapel berdasarkan isi Kolom A tersebut.

3. Sistem Absensi Cerdas (H, S, I, A)

Pada tab "Absensi", jangan mengetik manual! Cukup KLIK pada sel/kotak kehadiran untuk mengubah status siswa secara otomatis dari Hadir (Biru) -> Sakit (Kuning) -> Izin (Ungu) -> Alpa (Merah).

üìã Input Data Siswa Cepat

Paste (Tempel) daftar nama siswa Anda dari Microsoft Excel atau Notepad ke dalam kotak di bawah ini. Pastikan 1 baris = 1 nama siswa.
