Cara Memasukkan CP dan TP
  • Buka menu Setup CP & TP.
  • Ketikkan narasi lengkap Capaian Pembelajaran (CP) untuk fase yang Anda ajar.
  • Pada tabel Tujuan Pembelajaran, klik tombol âž• Tambah Bab/Materi.
  • Masukkan Nomor Bab, Judul Materi/Tujuan Pembelajaran, dan Alokasi Waktu (JP).
  • Setelah semua materi terisi, klik ðŸ’¾ Simpan Data CP & TP.
  • Catatan: Jika Anda Guru PAI SD, pilih kelas di dropdown "Muat Data Default PAI SD" lalu klik Load.
Cara Input Data Siswa via CSV
  • Siapkan Microsoft Excel. Buat 5 kolom dengan header: nisn, nama, jk, kelas, rombel.
  • Isi data siswa. Contoh: 001234, Budi Santoso, L, 3, 3A.
  • Save As â†’ CSV (Comma delimited) (*.csv).
  • Upload file CSV di tab Data Siswa.
Cara Mengatasi Hasil Cetak Terpotong
  • Klik tombol Cetak di dalam aplikasi (bukan Ctrl+P).
  • Pastikan Layout sesuai rekomendasi (Portrait/Landscape).
  • Aktifkan opsi "Background graphics" agar warna tabel muncul.
  • Ubah Margins ke Default atau None.
Cara Menggunakan Kalender
  • Kalender sudah memuat hari libur nasional baku secara otomatis.
  • Untuk hari libur tidak baku (Idul Fitri, Idul Adha, dll), tambahkan secara manual via form Tambah Libur Kustom.
  • Klik pada tanggal di kalender untuk melihat detail hari libur.
